#  مستندات پروژه TaskFlow

| شماره | عنوان مستند | توضیحات | مسیر فایل |

| 1 | طراحی پایگاه داده | شامل توضیح جداول مربوط به کشاورزان، محصولات، سفارش‌ها و خریداران است. | [database_design.md](./database_design.md) |

| 5 | راهنمای نصب و پیکربندی | شرح تنظیمات اولیه سرور، دیتابیس و نحوه اجرای پروژه در محیط توسعه و محصول. | [installation_guide.md](./installation_guide.md) |
